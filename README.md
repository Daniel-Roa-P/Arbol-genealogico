# Árbol genealógico

El siguiente aplicativo realizado en Prologo al hacer uso de la programación lógica se encarga de hacer consultas acerca de las relaciones existentes en un árbol genealógico de 4 generaciones, junto a saber si alguien es feliz dependiendo si está casado.
